# sample-node-web-api
A sample REST API running on Node, using Express and MongoDB.

## Install Dependencies

`npm install`

## Run

`node app.js`  

The API is available at: `http://localhost:3000/items`.

### Supported Actions

`GET /items`  
`GET /items/:id`  
`POST /items`  
`PUT /items/:id`  
`DELETE /items/:id`
